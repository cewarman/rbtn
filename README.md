# Rule-based text normalization system

## ✨ Features
消除文字的歧異，為文字轉語音系統的前置工作。<br>
如：<br>
&emsp;7/19是星期五  ->  七月十九號是星期五<br>
&emsp;2/3的人不喜歡香菜  ->  三分之二的人不喜歡香菜<br>
## 📦 Installation
請先安裝[OpenFst](https://www.openfst.org/twiki/bin/view/FST/WebHome)後再make
## 🚀 Quick Start
```terminal
./RBTN rules_config.txt < fin.txt
```
